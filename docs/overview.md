# 🚨 Fraud Detection Engine — Overview

## 📌 Introduction

Fraud Detection Engine is a mobile-first system designed to identify and explain digital fraud attempts such as phishing messages, scam links, and suspicious phone numbers.

The project focuses on combining **security, usability, and explainability**, making it accessible to non-technical users while maintaining a robust detection approach.

---

## 🎯 Objective

The main goal of this system is to:

- Detect potential fraud in real-time
- Analyze different types of input (messages, links, phone numbers, QR codes)
- Assign a risk score based on multiple factors
- Provide clear explanations to help users understand the risk

Unlike traditional security tools, this system emphasizes **user understanding**, not just detection.
